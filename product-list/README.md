## 总结
- 监听路由变化的方式
    - `<router-view :key="$route.path"></router-view>` 这种方式会让router-view 中的组件每次都重新渲染，无法记录里面的状态,比如p值等

- 共享一个window scroll事件，当一个unbind之后，后续的如何处理，`:key="$route.path`生产多个滚动监听，但是名字是一个，所以要为每个scroll单独命名一个名字。
